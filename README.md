# Журнал курильщика (SmokeFreeWay)

Персональный ассистент для отказа от курения. Помогает отслеживать прогресс, заменять ритуал курения полезными привычками и не терять мотивацию после срывов.

## Основные возможности
- Трекер чистых дней и сэкономленных денег
- SOS-помощник при тяге (дыхательные упражнения)
- Журнал срывов без обнуления прогресса
- Статистика восстановления организма

## Стек
- Flutter / Dart
- Hive (локальное хранилище)
- Тёмная тема в стиле Material Design

## Запуск
1. Установите Flutter SDK (>=3.0)
2. Клонируйте репозиторий
3. Выполните `flutter pub get`
4. Запустите `flutter run`
